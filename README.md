CLI tool for fastret-framework

> Give us a :star: if you like our work :heart:

## Install

```shell
$ npm i -g fastret-cli
```

## Usage

```shell
$ fastret tests/data.json

# Output
# ✔ Passed: GET API TEST
# ✖ Failed: GET API WRONG TEST => ()
```
